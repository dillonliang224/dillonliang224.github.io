Java设置编码

2016-03-08 • 标签: Java Encoding

今天在本地创建一个properties的文件，里面写了中文，保存的时候报了一个错误：

这是因为编码问题导致的。

解决方法：

  • Window->Preferences->General->Content Types->Text->Java Properties File设置Default encoding,把ISO-8859-1改为UTF-8 然后update.

  • Window -> Preferences -> General -> Workspace -> Text file encoding，选择Other选项的UTF-8

  • Window -> Preferences -> Web -> JSP Files -> Encoding选择ISO 10646/Unicode(UTF-8)，点Apply按钮

我们把Eclipse上的所有位置设为UTF-8就可以了，只要保存文件出现encoding问题，就看下这三个地方。
