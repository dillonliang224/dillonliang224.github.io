编译Java代码的时候有时候会发生如下错误：

这个错误是JDK版本过低的原因。

我测试的时候本机使用的是jdk1.6，项目编译用1.7，导致异常。

各JDK版本对应的错误编号：

所以我换了个1.7jdk重新编译就好了。
步骤如下：

  1. 修改Eclipse的JDK版本

    Window -> Preference -> Java -> Installed JREs

    选择一个高版本的，如果没有下载一个了。

  2. 修改项目的编译版本
    选中项目，alt + enter，找到Java Compiler，重新选中编译
